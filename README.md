# BasicRetroLinux
A rice for hyprland that is based off of 80s-90s box computer ui.
HEAVILY INSPIRED BY DIINKI'S RETRO THEME, CHECK THEIRS OUT!!!
2 wallpapers inside are from diinki! credits to her (the wallpapers are linux.png and the redanime)

Basic tutorial:
The wallpaper selector uses swww, change it by going into rofi/wallpaper-launcher.sh and at the bottom, change whatever to use your wallpaper selector.

INSTALL:
clone it by using:
git clone https://github.com/vinsakkers-coder/BasicRetroLinux.git cd BasicRetroLinux
and then run the install.sh inside of terminal using
sh ./install.sh
